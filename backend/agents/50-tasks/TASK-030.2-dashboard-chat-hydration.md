# TASK-030.2 대시보드 채팅 응답 수신/재로딩 안정화

## 목표
- 대시보드 채팅이 다른 페이지로 이동하거나 새로고침해도 백엔드 응답을 정상적으로 반영하고, 이전 메시지가 다시 스트리밍되지 않도록 한다.

## 완료 항목
- AI 메시지를 저장/복원할 때 스트리밍 플래그를 제거해 재방문 시 즉시 전체 답변이 보이도록 수정.
- 백엔드 응답 필드를 `answer`, `response.answer`, `data.answer` 순으로 안전하게 파싱해 로딩이 멈추는 문제 방지.
- 로컬 스토리지에 기록되는 채팅 상태를 정상화(normalize)하여 탭 전환/리로드 이후에도 동일한 메시지와 로딩 상태가 유지되도록 처리.
