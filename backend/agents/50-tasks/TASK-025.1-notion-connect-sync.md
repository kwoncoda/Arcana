## 요청 사항
- 노션 연동을 마친 후 연동 페이지에서 성공 메시지(“노션이 연동되었습니다”)가 표시되도록 개선합니다.
- 노션 OAuth 콜백 완료 직후 해당 데이터 소스의 지식 베이스(RAG 인덱스) 갱신을 자동으로 실행합니다.

## 진행 기록
- 초기 상태 기록.
- 프론트 OAuth 콜백에서 노션 연동 직후 `/notion/pages/pull`을 자동 호출해 지식 베이스를 최신화하도록 추가했습니다.
- 연동 완료 후 연동 페이지로 이동하며 "노션이 연동되었습니다" 메시지와 동기화 실패 안내가 표시되도록 개선했습니다.

## 결과
- 사용자가 노션 인증을 마치면 즉시 RAG 동기화를 시도해 데이터가 자동 갱신됩니다.
- 연동 페이지에서 성공 메시지와 동기화 상태를 확인할 수 있어 UX가 향상되었습니다.
