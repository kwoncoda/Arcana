## 요청 사항
- OAuth 연동 직후 백그라운드에서 즉시 지식 베이스 갱신이 실행되어 인증 흐름에 영향을 준다.
- 연동 완료 후 대시보드에서 동기화 진행을 전면 오버레이로 안내하며, 수동 갱신 버튼을 눌러도 동일한 차단 오버레이를 보여준다.

## 진행 기록
- 2025-11-22: 초기 상태 기록. 요구사항 확인 및 대시보드 동기화 흐름 파악.
- 2025-11-22: OAuth 콜백의 즉시 동기화 호출을 제거하고, 대시보드로 리디렉션 후 오버레이 상태에서 동기화를 트리거하도록 설계.
- 2025-11-22: 대시보드의 수동 동기화 버튼에도 동일한 풀스크린 오버레이를 적용해 다른 페이지 이동을 차단하도록 구현.

## 결과
- OAuth 인증 이후에는 대시보드에서 전면 오버레이를 띄운 상태로 지식 베이스 갱신을 진행하도록 변경.
- 지식 베이스 갱신 버튼 실행 시에도 동일한 차단 오버레이가 표시되어 동기화 완료 전 다른 작업이 불가능하다.
