# TASK-030.3 대시보드 채팅 백그라운드 재시도 및 질문 표시

## 목표
- 페이지 이동/새로고침으로 요청이 끊겨도 대시보드에 복귀하면 동일한 질문을 자동 재시도해 답변을 복원한다.
- 전송 직후 로딩 중에도 사용자가 입력한 질문이 즉시 채팅 타임라인에 표시되도록 한다.

## 완료 항목
- 대시보드 채팅 상태를 복원할 때 진행 중이던 질문이 있으면 동일 쿼리로 재호출하도록 재시도 루프를 추가.
- 로컬 스토리지 상태에 사용자 질문이 누락된 경우 자동으로 삽입해 로딩 상태와 함께 표시.
- 검색 요청 처리 로직을 별도 함수로 분리해 전송/재시도 경로에서 동일한 예외 처리와 응답 파싱을 공유.
