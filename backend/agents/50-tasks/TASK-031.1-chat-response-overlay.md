## 요청 사항
- AI 챗봇 질문을 전송한 후 백엔드 응답을 기다리는 동안 전체 화면 오버레이를 띄워 다른 페이지 이동을 막고 "답변 중입니다" 메시지를 노출한다.
- 지식 베이스 갱신 로직의 전면 로딩 UX와 일관되게 설계한다.

## 작업 내역
- MainDashboard에 `isChatLoading` 상태를 감지하는 전면 오버레이를 추가하고, 스피너와 "답변 중입니다" 안내 문구를 배치했다.
- 오버레이 내부에 "중단하기" 버튼을 두어 백엔드 요청을 즉시 취소하고 기존 정지 핸들러(`handleStopMessage`)를 호출하도록 연결했다.
- 오버레이 z-index를 지식 베이스 갱신 오버레이보다 높게 설정해 질문 처리 중 다른 화면 요소와 상호작용하지 못하도록 차단했다.

## 테스트
- 수동: 질문 전송 시 오버레이 표시 및 "중단하기" 버튼으로 요청 취소 여부 확인.
